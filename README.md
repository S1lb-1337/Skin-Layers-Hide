> ### "Skin layers hide - Groups" ###

>  "Body on/off"
> 
>  "Pants on/off"

>  "Jacket on/off"
> 
>  "Right sleeve on/off"
> 
>  "Left sleeve on/off"
> 
>  "Right pains leg on/off"
> 
>  "Left pains leg on/off"
> 
>  "Cape on/off"
