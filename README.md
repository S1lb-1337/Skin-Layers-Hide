# skin-layers-hide
Switch skin layers by hotkeys

To change the bind, go to the settings and there will be categories.

"Skin layers hide - All"
- All
- All (The cape does not turn off)

"Skin layers hide - Groups"
- Top
- Bottom

"Skin layers hide - Parts"
- Hat
- Jacket
- Right hand
- Left hand
- Right leg 
- Left leg
- Cape
